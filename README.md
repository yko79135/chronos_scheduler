# Chronos Scheduler

Excel(`.xlsx`)에 입력된 학생, 학년, 과목, 교사, 주당 시수 정보를 브라우저에서 직접 읽어 교사/학년/학생 시간표를 생성하는 TypeScript 웹앱입니다.

## 실행

```bash
npm run build
npm run dev
```

브라우저에서 `http://localhost:5173`을 열고 `스케줄링_2026가을.xlsx` 같은 원본 Excel 파일을 업로드합니다. CSV 변환은 필요하지 않습니다.

## 구현 범위

- `학생별 학년`, `교사별`, `학년별`, `시간표`, `과목배분` 시트 목록 및 반복 표 구조 분석
- 학생/학년/교사/과목/수업요구 정규화 모델 변환
- 학년 표현 파서: `G1`, `1-3학년`, `G7E+G7K`, `G4-12`, `All` 등 지원
- MRV 기반 백트래킹 솔버
- 하드 제약: 교사/학년/교실 이중 배정 방지, 연속수업, 점심시간 교차 금지, 방과후 8교시 제한, 공유수업 동시 점유
- 불가능한 수업은 미배정 및 진단 메시지로 표시
- 대시보드, 검증 경고, 학년별/교사별 결과표, 저장/불러오기
- Excel 내보내기와 인쇄/PDF 출력

## 파일 형식

- `교사별`: 교사명 아래 `학년 / 수업 / 주당 시간 / 주 횟수` 4열 반복
- `학년별`: 학년명 아래 `수업 / 시간 / 횟수` 3열 + 빈 열 반복
- `과목배분`: 행은 과목, 열은 학년, 셀은 담당 교사명 또는 이니셜
- `학생별 학년`: 학생 이름, 최종 학년, 과목별 참여/제외 정보를 읽습니다.

## 제약조건

완전 배정이 불가능하면 제약을 몰래 무시하지 않고 미배정 목록에 이유를 표시합니다. 총 시수가 횟수로 나누어떨어지지 않는 경우 자동 후보 블록을 만들고 경고를 표시합니다.
